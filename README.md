# HTML, CSS, and JavaScript Projects

This workspace contains several projects to enhance HTML and CSS skills and learn JavaScript for DOM manipulation.

## Projects

### 1. Contact Form
- **Location**: `contact-form/`
- **Description**: A contact form with HTML form elements styled with CSS, and JavaScript validation for required fields and email format.
- **Files**:
  - `index.html`: The HTML structure of the form.
  - `styles.css`: CSS styling for the form.
  - `script.js`: JavaScript for form validation.

### 2. Responsive Layout
- **Location**: `responsive-layout/`
- **Description**: A webpage demonstrating responsive design using Flexbox for navigation and CSS Grid for content areas, with media queries for different screen sizes.
- **Files**:
  - `index.html`: The HTML structure with header, nav, main content, and footer.
  - `styles.css`: CSS with Flexbox, Grid, and media queries.

### 3. Dynamic To-Do List
- **Location**: `todo-list/`
- **Description**: A to-do list application using JavaScript for DOM manipulation to add, edit, mark as complete, and remove tasks.
- **Files**:
  - `index.html`: The HTML structure with input and list.
  - `styles.css`: CSS styling for the list and buttons.
  - `script.js`: JavaScript for adding, editing, completing, and deleting tasks.

## How to Run
Open the `index.html` file in any modern web browser for each project. No server required as they are static HTML pages.

## Learning Objectives
- Build forms with various input types and style them with CSS.
- Implement client-side form validation with JavaScript.
- Create responsive layouts using Flexbox and CSS Grid with media queries.
- Manipulate the DOM with JavaScript to create dynamic user interfaces.
